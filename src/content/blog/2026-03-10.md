## 開發環境

今天花了一點時間把 Arch Linux 的開發環境整理好：

- 確認 Node.js (via nvm) 與 pnpm 都是最新版本
- 設定好 Git 的 GPG signing
- 把 Neovim 的 LSP 設定同步到新機器

## AI Agent 框架研究

初步比較了幾個框架：

1. **LangGraph** — 適合需要複雜 state machine 的場景
2. **CrewAI** — 多 Agent 協作的抽象做得不錯
3. **Claude Agent SDK** — Anthropic 官方出的，API 整合最直接

目前傾向先用 Claude Agent SDK 做 prototype，之後再看需不需要換。
